## Authentication
This provider requires authentication.
- **project_id**: Google Cloud project ID (required: True, sensitive: False)
- **subscription**: PubSub subscription name (required: True, sensitive: False)
- **service_account_json**: Service account JSON with pubsub subscriber role (required: False, sensitive: True)

Certain scopes may be required to perform specific actions or queries via the provider. Below is a summary of relevant scopes and their use cases:
- **roles/pubsub.subscriber**: Read access to PubSub subscription (mandatory)
