## Authentication
This provider requires authentication.
- **service_account_json**: A service account JSON with pubsub subscriber role (required: False, sensitive: True)
- **subscription**: Pub/Sub subscription name or full path (required: True, sensitive: False)
- **project_id**: Google Cloud project ID (required if subscription is not full path) (required: False, sensitive: False)
- **topic**: Pub/Sub topic name or full path used for publishing (required: False, sensitive: False)

Certain scopes may be required to perform specific actions or queries via the provider. Below is a summary of relevant scopes and their use cases:
- **roles/pubsub.subscriber**: Read access to Pub/Sub subscriptions (mandatory)
