## Connecting with the Provider

1. Create or choose a Pub/Sub subscription that receives your alert messages.
2. Provide the subscription name or full path in the provider configuration.
3. If not using Workload Identity, supply a service account JSON with the `roles/pubsub.subscriber` role.
